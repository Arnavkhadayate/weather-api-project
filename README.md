# weather-api-project
A weather application built with JavaScript and Weather API integration.

## Features

- Search weather by city name
- Display temperature
- Display weather conditions
- Error handling for invalid city names

## Technologies Used

- HTML
- CSS
- JavaScript
- Weather API
